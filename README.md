## Getting Started

1. Clone the repository

```bash
git clone [repository-url]
```

2. Install dependencies

```bash
cd frontend
npm install
```

3. Set up environment variables

```bash
cp .env.example .env
```

4. Start the development server

```bash
npm run dev
```

## Environment Variables

```env
VITE_API_URL=http://localhost:5000
```
